# FIMpad Themes

Place your caret within in a config tag below and press Alt+= (or use the menu entry at AI -> Apply Config Tag) to apply it. This will change the colors in your text editor.

---

## The Beach

A lively day at the beach.

```
[[[{
"fg": "#d8d8d8",
"bg": "#384868",
"highlight1": "#c800c8",
"highlight2": "#dd9500"
}]]]
```
